以下仅摘记所遇有关Linux命令
<!--more-->
1. mkdir xx
创建名为xx的文件夹
2. cd xx
跳转到xx目录
3. pwd
显示当前目录路径
4. ls -a 
列出文件下所有的文件，包括以“.“开头的隐藏文件（linux下文件隐藏文件是以.开头的，如果存在..代表存在着父目录）
5. cat xx.txt
显示当前文件内容
6. rm xx.txt
删除文件
7. vim xx.txt
（创建）编辑文件
Esc : wq 保存退出
